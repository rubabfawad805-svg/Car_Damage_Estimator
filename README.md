# Car_Damage_Estimator


A web application that allows users to upload a car image and get **automated damage analysis** and **repair cost estimation** using AI.

---

## **Features**

- 📤 Upload car images (PNG, JPG, JPEG)
- 🔍 AI-powered **damage analysis**
  - Identify visible damage (dents, scratches, broken parts)
  - Specify damage location (front bumper, doors, hood, etc.)
  - Assess severity (minor, moderate, severe)
- 💰 AI-powered **cost estimation**
  - Itemized repair costs (parts + labor)
  - Total estimated cost in USD
  - Damage category (Minor / Moderate / Major)
  - Estimated repair time
  - Recommendations (repair vs replace)
- 📋 Generate **full damage report** for download
- ⚡ Simple, interactive Streamlit UI
